实习目的

学习掌握Qimera软件处理多波束数据的工作流程，并用Qimer软件对一个航次的多波束数据进行处理。

数据概述

  • 数据采集航次的科考船为：Sumner

  • 航次时间：2010/08/06——2010/09/05

  • 仪器：Simrad EM122

  • 测线数：216

  • 调查区域：

处理流程

打开Qimera新建项目

Project->New Project

添加资源文件

Source->Add Raw Sonar File 将Simrad EM122采集的*.ALL原始数据导入到软件中，进行数据转换。

Qimera将尝试自动检测和提取任何可用信息。提取的信息包括船舶配置、位置、航向、运动、声速、潮汐或GPS/GNSS高度校正（如果文件格式支持并在文件中可用）

创建动态曲面

Dynamic Surface->Create Dynamic Surface

编辑水深测量数据

可用工具：

  • 3D Editor
  • Slice Editor
  • Swath Editor
  • Spline Filtering

利用3D Editor进行编辑

利用Swath Editor进行编辑

产品导出

Layer->Export Surface

实习收获

通过《多波束数据处理》课程的实习，我基本掌握了利用Qimera软件处理多波束数据了流程，并且可以对原始的多波束数据进行简单的处理，得到表面或者点云产品，为接下来相关方面研究提供了数字资料。
