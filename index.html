实习目的

学习掌握Qimera软件处理多波束数据的工作流程，并用Qimer软件对一个航次的多波束数据进行处理。

数据概述

  • 数据采集航次的科考船为：Sumner

  • 航次时间：2010/08/06——2010/09/05

  • 仪器：Simrad EM122

  • 测线数：216

  • 调查区域：

处理流程

打开Qimera新建项目

Project->New Project

添加资源文件

Source->Add Raw Sonar File 将Simrad EM122采集的*.ALL原始数据导入到软件中，进行数据转换。

Qimera将尝试自动检测和提取任何可用信息。提取的信息包括船舶配置、位置、航向、运动、声速、潮汐或GPS/GNSS高度校正（如果文件格式支持并在文件中可用）

创建动态曲面

Dynamic Surface->Create Dynamic Surface

编辑水深测量数据

可用工具：

  • 3D Editor
  • Slice Editor
  • Swath Editor
  • Spline Filtering

利用3D Editor进行编辑

利用Swath Editor进行编辑

产品导出

Layer->Export Surface

实习收获

通过《多波束数据处理》课程的实习，我基本掌握了利用Qimera软件处理多波束数据了流程，并且可以对原始的多波束数据进行简单的处理，得到表面或者点云产品，为接下来相关方面研究提供了数字资料。

创建船舶文件(针对本次实习数据)

  1. 在与项目文件夹同级的路径下建立VesselConfig文件夹，船舶文件将被自动保存在此文件夹下进行管理。

  2. Tools->Editors->Vessel->New进入船舶文件编辑向导。

    • Step-1:创建日期一定要在数据采集日期之前、命名不能有空格

    • Step-2:测量类型：Multibeam

    • Step-3:探头：1 波束数：432 多波束型号：EM122

    • 其它默认

      船舶吃水问题？

      –编辑Dynamic Draft校正船舶动态吃水变化。

      –编辑Waterline Height校正船舶静态吃水变化。

新建工程

  1. 新建工程前，确保同级目录下有建好的船型文件

  2. New Project打开一个新建工程的向导。(需要提前连接本地一个仓库：在Step1空白处右键，选择Connect to)

  3. Step1：为工程命名，选择船型文件，工程日期

  4. Step2-4：默认即可

原始数据转换

  1. 利用File->import->Conversion Wizard向导来进行数据转换
  2. Step1：本次实习所需要转换的数据格式为：Simrad（具体所支持格式可以参见软件帮助-用户手册）


3. Step3：选择原始数据
4. Step4：选择数据转换到的项目，船型文件和日期
5. Step9：点击转化按钮（其余步骤均默认）

几点说明

  • Step2各选项含义：
    • Create new survey lines：如果项目中存在测线数据，将覆盖存在数据
    • Update existing survey lines：选择在现有项目中替换哪种数据
    • Carry Over Raw Data Files：将原始数据文件复制到已处理的文件夹
    • Search for Files with Temporal Overlap：扫描包含选定要转换的文件的整个目录，此扫描检查具有时间重叠的导航或运动数据，这样做是为了在采集软件将一条测量线分成多条的情况下，正确地同步参考时间戳

错误经验

笔者按照工作流程在进行数据转换时始终无法成功？

–查阅转换报告后，发现犯下的低级错误

Error description = All timestamps for the given sensor are after the requested time.
Error: Unable to construct ellipsoid specified in the vessel configuration.

错误提示船型文件时间早于数据时间！！！从整个报告中也可以看出，CARIS软件基于C++构建，C++中进行宽窄字符的转换不是一件非常容易的事情吗？吐槽垃圾的中文输出乱码问题！！！

技巧

  • Save Session保存当前工作状态，方便下次工作
  • Tools->Options->File Location自定义默认文件路径，方便数据导入

问题

因《多波束数据处理》课程的需要，需要从https://www.ngdc.noaa.gov/mgg/bathymetry/relief.html 网站下载课程所需数据。但是多波束数据文件多，数据量大，存在用浏览器下载不稳定，网站数据文件打包易中断等问题。

解决办法

用python爬取网页下载链接后，用破解版迅雷进行批量下载。
